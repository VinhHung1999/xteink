### Email Dos and Don'ts

| Do | Don't |
|----|-------|
| Write like a personal letter | Write like a corporate announcement |
| One primary CTA per email | Multiple competing CTAs |
| Use Cormorant Garamond for headings (with Georgia fallback) | Use web-only fonts without fallbacks |
| Keep emails short (300 words max for newsletter) | Write essay-length emails |
| Include an unsubscribe link (legally required + soul-aligned: respect choice) | Hide the unsubscribe link |
| Send max 1x/week | Send daily promotional emails |

## Performance Standards

Per the E-Ink / Paper style, performance is excellent:

| Metric | Target | Notes |
|--------|--------|-------|
| **First Contentful Paint** | < 1.5s | Fast â€” minimal assets |
| **Largest Contentful Paint** | < 2.5s | Hero image optimized |
| **Cumulative Layout Shift** | < 0.1 | No layout-shifting animations |
| **Total page weight** | < 1.5MB | Compressed images, minimal JS |
| **Image format** | WebP with JPG fallback | E-Ink style = low-complexity images compress well |
| **Font loading** | `font-display: swap` | No FOIT (Flash of Invisible Text) |
| **Lazy loading** | Below-fold images | `loading="lazy"` attribute |
